This page runs the real, deployed version of the Skill Stork assistant — same backend, same knowledge base, same design that will appear on the actual website.

To try it: click the round logo bubble in the bottom-right corner.

To embed it on the real site: copy the one <script> tag from README.md onto any page on skillstork.org — no other setup needed there.

To edit what it knows: open server.js and update the SYSTEM_PROMPT — fees, staff, calendar, or anything else the school provides officially.
